# Halogenating agents (Cl / Br / I introductions)

This page collects the “go-to” reagents for introducing halogens, especially in NCERT-style organic transformations.

## Converting alcohols / acids to alkyl acyl halides

| Transformation | Reagent(s) | Notes |
|---|---|---|
| \(\mathrm{ROH \rightarrow RCl}\) | **\(\mathrm{SOCl_2}\)** (often pyridine) | Very common: gaseous byproducts (\(\mathrm{SO_2}\), \(\mathrm{HCl}\)) drive reaction. |
| \(\mathrm{ROH \rightarrow RCl}\) | \(\mathrm{PCl_3}\), \(\mathrm{PCl_5}\) | Alternative chlorinating agents. |
| \(\mathrm{ROH \rightarrow RBr}\) | \(\mathrm{PBr_3}\) | Often cleaner than \(\mathrm{HBr}\) for some substrates. |
| \(\mathrm{RCOOH \rightarrow RCOCl}\) | **\(\mathrm{SOCl_2}\)** / \(\mathrm{PCl_5}\) | Acid chlorides: key acylation intermediates. |

## Aromatic halogenation (electrophilic substitution)

- **\(\mathrm{Cl_2/FeCl_3}\)**: chlorination of benzene/arenes.
- **\(\mathrm{Br_2/FeBr_3}\)**: bromination of benzene/arenes.

## Allylic / benzylic bromination (common “special case”)

- **NBS** (N-bromosuccinimide), light/peroxide: tends to favor allylic/benzylic bromination (radical conditions).

## Diazonium → aryl halide (Sandmeyer-type)

- \(\mathrm{ArN_2^+Cl^- \xrightarrow{CuCl/HCl} ArCl}\)
- \(\mathrm{ArN_2^+Cl^- \xrightarrow{CuBr/HBr} ArBr}\)
- \(\mathrm{ArN_2^+Cl^- \xrightarrow{KI} ArI}\)

## “What not to mix up” reminders

- \(\mathrm{Cl_2/FeCl_3}\) is **aromatic EAS** halogenation; it is not the same use-case as \(\mathrm{SOCl_2}\) (alcohol/acid chlorination).
